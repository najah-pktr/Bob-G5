# 🥔 Bob-G5: Artificial Non Intelligent
### By Potato Labs

Bob-G5 is a tiny, from-scratch Transformer AI built entirely with PyTorch to teach people how Large Language Models actually work. No cheating, no massive pre-trained APIs—just pure math, attention mechanisms, and a lot of potato power.

![Python](https://img.shields.io/badge/Python-3.10+-blue.svg)
![PyTorch](https://img.shields.io/badge/PyTorch-2.0+-red.svg)
![License](https://img.shields.io/badge/License-MIT-green.svg)

### 🧠 The Philosophy
Big AI companies hide their code behind APIs. At Potato Labs, we believe the best way to learn AI is to build it from scratch—even if it's a little bit "Non Intelligent." 

Bob-G5 is a **636,423 parameter** language model. He uses:
- Token + Positional Embeddings
- Causal Self-Attention (so he doesn't peek at the future)
- Transformer Encoder blocks
- Greedy Decoding with Anti-Loop mechanics

### 🚀 Quickstart

1. Clone the repo:
```bash
git clone https://github.com/najah-pktr/Bob-G5.git
cd Bob-G5
```

2. Install requirements:
```bash
pip install torch
```

3. Run Bob:
```bash
python ani.py
```

### 💬 How to talk to Bob
Bob is a tiny AI. To get the best results, talk to him in **lowercase without punctuation**.
- ✅ `who are you`
- ✅ `what is the capital of france`
- ✅ `tell me a joke`
- ❌ `Hello, who are you?!` (He gets confused by commas and capital letters)
